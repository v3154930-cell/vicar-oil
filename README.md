# Vicar Oil - Design Assets

Репозиторий с дизайн-активами для обновления лендинга компании "Викар-Ойл".

## Структура

```
vicar-oil/
├── svg-icons.html      # SVG иконки для типов топлива
├── button-styles.css   # CSS стили для CTA-кнопок
├── svg-patterns.html   # SVG паттерны для фонов
├── STOCK-PHOTOS.md     # Ссылки на стоковые фото
└── README.md           # Этот файл
```

## Использование

### 1. SVG Иконки
Откройте `svg-icons.html` в браузере для просмотра иконок:
- Бензин (Gasoline)
- Дизель (Diesel)  
- Керосин (Kerosene)

### 2. CSS Кнопки
Подключите `button-styles.css` в вашем HTML:
```html
<link rel="stylesheet" href="button-styles.css">

<!-- Primary CTA -->
<a href="#" class="btn-cta-primary">
  Получить предложение
  <span class="arrow">→</span>
</a>

<!-- Secondary CTA -->
<a href="#" class="btn-cta-secondary">
  Смотреть услуги
</a>
```

### 3. SVG Паттерны
Откройте `svg-patterns.html` для просмотра паттернов:
- Oil Droplet (капли нефти)
- Pipeline (трубопроводы)
- Geometric Grid
- Hexagon Industrial
- Dot Matrix
- Wave Flow

### 4. Фото
См. `STOCK-PHOTOS.md` для подборки стоковых фото рефайнри.
